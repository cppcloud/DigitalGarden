## 基本原理
Linux虚拟内存范围分为内核空间部分、用户空间部分和每个进程中用于捕获NULL指针的不可访问部分。我使用的是GentooLinux，其中这个不可访问的部分是从 `0x0000` to `0xFFFF` ，通过以下命令可以看出
```bash
cat  /proc/sys/vm/mmap_min_addr
65536
```
